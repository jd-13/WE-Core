Detailed Description

template<typename BASE>
class WECore::LookAndFeelMixins::PopupMenuV2< BASE >

V2 (December 2018) style lookandfeel popup ment mixin.

Uses the following colours:

  1. ** PopupMenu::highlightedBackgroundColourId **
  2. ** PopupMenu::highlightedTextColourId **
  3. ** PopupMenu::textColourId **
