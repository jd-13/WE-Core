Detailed Description

template<typename T, size_t NUM_FORMANTS>
class WECore::Songbird::SongbirdFormantFilter< T, NUM_FORMANTS >

A class containing a vector of bandpass filters to produce a vowel sound.

Supports only mono audio processing. For stereo processing, you must create two objects of this type. (Do not reuse a single object for both channels)

To use this class, simply call setFormants, reset, and process as necessary.

See also
setFormants - must be called before performing any processing
Formant - Formant objects are required for operation of this class
