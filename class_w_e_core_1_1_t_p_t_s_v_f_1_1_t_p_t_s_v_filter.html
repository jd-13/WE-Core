Detailed Description

template<typename T>
class WECore::TPTSVF::TPTSVFilter< T >

A state variable filter from a topology-preserving transform.

To use this class, simply call reset, and the process methods as necessary, using the provided getter and setter methods to manipulate parameters.

Internally relies on the parameters provided in TPTSVFilterParameters.h

Based on a talk given by Ivan Cohen: https://www.youtube.com/watch?v=esjHXGPyrhg
