Detailed Description

template<typename BASE>
class WECore::LookAndFeelMixins::ButtonV2< BASE >

V2 (December 2018) style lookandfeel button mixin.

Uses the following colours:

  1. ** TextButton::buttonOnColourId **
  2. ** TextButton::buttonColourId **
  3. ** TextButton::textColourOnId **
  4. ** TextButton::textColourOffId **
